passkey-demos

A collection of demos for passkey / WebAuthn features. Most of these used to live on glitch and need a new home.

  • Back-forward cache

    This demo has a button that starts a WebAuthn request and a few seconds later navigates the page. It helps debug bugs with bfcache.

  • Conditional UI

    This was the first website showcasing Conditional UI, i.e. the integration between passkeys and autofill. It's still useful as a barebones test site.

  • Conditional UI with a button

    This is very similar to the Conditional UI demo site, but it has a button that triggers the WebAuthn call. It's useful to test what happens when the user opened autofill and *then* the browser received the call.

  • Large blob

    This is a short demo of the large blob feature that lets the user store and retrieve a large blob associated to a WebAuthn credential.

  • Signal API

    This is a demo that showcases all three (at the time of writing) signal* methods.

  • Cigüeña

    This demo is intended to be a test site for internationalized domain names. However, on GitHub pages I'm not sure how to put it in a custom subdomain, so it's not very useful at the moment.
